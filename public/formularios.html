Formularios

Para maquetar formularios utilice EDgrid, aplicando la clase ed-container a la etiqueta formy encerrando los componentes del formulario dentro de los ed-item que se requieran.

Se recomienda usar label o placeholder, pero no ambos.

Los inputs de tipo radio o checbox deben ser hijos directos de la etiqueta label

Use la clase form__item o form-item para encerrar los campos y crear las separaciones verticales.
