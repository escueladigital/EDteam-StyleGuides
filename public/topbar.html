Top bar

La barra superior (topbar) se usa para mostrar breadcrumbs, filtros o botones con llamadas a la acción.

La estructura recomendada es la siguiente (el CSS está definido para esa estructura), aunque con EDgrid puede hacerse variaciones.

<div class="topbar">
  <div class="ed-container">
    <div class="ed-item l-50">
      <p>Características > Topbar</p>
    </div>
    <div class="ed-item l-50 main-end">
      <input placeholder="Encuentra algo">
      <a class="button--cta" href="#">Modelo de boton</a>
    </div>
  </div>
</div>

Para cambiar de color a la topbar use las siguientes clases .topbar--secondary, .topbar--tertiary, .topbar--accent o .topbar--dark.
